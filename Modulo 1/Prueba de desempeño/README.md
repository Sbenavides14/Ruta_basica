# Sistema de gestion de estudiantes

## Descripcion

Este programa es un sistema de gestion de estudiantes, donde el usuario podra hacer seguimiento y administrar una lista
de estudiantes de acuerdo a sus necesidades.

Podra registrar, consultar, buscar, actualizar y eliminar a los estudiantes de acuerdo a sus necesidades mediante un menu interactivo e intuitivo.

## Como funciona:

1. Al ejecutar el programa el usuario se encuentra inicialmente con el menu de opciones anteriormente mencionadas. Donde el usuario podra escoger que accion desea realizar.
2. Una vez la opcion elegida por el usuario es validada, el programa ejecuta la accion, si la opcion no es validad, el programa solicitara nuevamente la opcion.
3. Cada cambio efectuado por el usuario como registro, actualizacion, eliminacion se veran reflejados de manera directa en la lista donde se almacenan los estudiantes.
4. Una vez el programa termina de ejecutar una de las funciones, regresa nuevamente al menu interactivo esperando una nueva instruccion.
Si el usuario decide culminar la ejecuion de programa, solo debera elejir la opcion de "Salir" en el menu, y el programa automaticamente detendra la ejecucion.

## Estado
El programa actualmente ejecuta el programa de manera ininterrumpida, ejecutando y validando cada funcion y dato de manera efectiva. El programa
tambien esta modularizado  por funciones y debidamente comentado para posteriores revisiones. Aunque el programa actualmente esta se encuentra construido de manera sencilla, es funcional y puede ser optimizado y reforzado mas a futuro.
